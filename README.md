# Pythonを使ったGaborウェーブレット変換ツール

# 使っているツール達
Python 2.7
PyQt4
PyOpenGL
Numpy
Scipy

# モチベーション
MatplotLibに頼らない

オーディオデータを読み込む事ができる。
